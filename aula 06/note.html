Seletores

Seguindo com o conteúdo do Khan Academy. Essa parte é sobre selecionar elementos no HTML. Parece fácil, mas tem situações do tipo: quero selecionar os elementos pares de uma lista com 10 elementos. Quero selecionar o "irmão" do elemento div, etc. O CSS dá suporte para você selecionar os elementos de muitas formas diferentes, nessa aula você vai entender as seleções que você vai mais usar no dia a dia, mas saiba que quando precisar de alguma seleção mais complexa o CSS vai te ajudar.
